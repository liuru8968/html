CSS基本語法

插入css檔案:

先建立css資料夾,在建css檔

link rel="stylesheet"href="CSS/檔名.css"

CSS結構:

選擇器{

屬性1:設定值;

屬性2:設定值;

}

  1. 文字顏色=color:顏色;
  2. 文字樣式(font)
    • 大小=font-size:數字px;

      • 數字10px

    • 字型=font-family:字型px;

      • 標楷體

    • 斜線=font-style:italic(斜線);

      • 斜線

    • 粗體=font-weight:bold(粗體);

      • 粗體

  3. 文字段落(text)
    • 對齊=text-align:左、右、置中(英文);

      • 我是置中

    • 縮排=text-indent:數字px;

      • 縮60px

    • 線條=text-decoration:underline(加底線);

      • 整段底線

  4. 文字行高=line-height:數字;

    5. 2倍

  5. 字元間距=letter-spacing:數字px

    6. 5px的字元間距
